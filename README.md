=Краткое описание основных функций программы
    Система реализации онлайн тестирования. Например, социологического опроса.

==Функциональность

* Анализ ответов на вопросы в опросниках/тестах
* Администрирование пользователей
* Администрирование тестов/опросников
* Самостоятельная регистрация пользователей
* Обнуление результатов
